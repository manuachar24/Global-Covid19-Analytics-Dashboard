# Global COVID-19 Analytics Dashboard | Power BI

## 📌 Overview
This project analyzes global COVID-19 data to understand trends across countries and continents.

---

## 🛠️ Tools & Technologies
- Python (Jupyter Notebook) → Data Cleaning
- MySQL Workbench → Data Analysis
- Power BI → Visualization Dashboard
- DAX → KPI and rate calculations

---

## 🔄 Workflow

1. Data Cleaning (Jupyter Notebook)
   - cleaned raw COVID dataset
   - handled null values
   - standardized columns

2. SQL Analysis (MySQL)
   - identified top affected countries
   - analyzed death and recovery trends
   - continent-wise comparisons

3. Dashboard (Power BI)
   - created KPI cards (Cases, Deaths, Recovery Rate)
   - built insight cards (Top Country, Death Rate, Recovery)
   - designed comparative visuals

---

## 📊 Key Insights
- Total Cases: 19M
- Total Deaths: 792K
- Recovery Rate: 63%
- Most Affected Country: USA
- Highest Death Rate Country: Vatican City
- Highest Recovery Country: Dominica
